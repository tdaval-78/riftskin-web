Guide installation LoL skins custom sur Mac

Comment installer des skins custom League of Legends (LoL) sur Mac

Ce guide explique comment installer RIFTSKIN sur macOS, comment commencer en gratuit, comment importer vos skins custom et quand il devient utile de passer au premium.

Étape 1: télécharger RIFTSKIN sur Mac

Commencez par récupérer l'installeur macOS depuis la page de téléchargement officielle. C'est la base du workflow pour utiliser des skins custom League of Legends sur Mac avec RIFTSKIN.

Télécharger RIFTSKIN

Étape 2: installer l'application

Ouvrez l'installeur, autorisez-le dans macOS si nécessaire, puis terminez l'installation. Une fois RIFTSKIN installé dans Applications, vous pouvez lancer le logiciel comme n'importe quelle autre application Mac.

  • Téléchargez l'installeur
  • Autorisez-le dans Confidentialité et sécurité si macOS le bloque
  • Terminez l'installation
  • Lancez RIFTSKIN depuis Applications

Étape 3: commencer en gratuit

Le mode gratuit suffit pour démarrer. Il permet d'importer vos skins custom, de les organiser et d'utiliser le cœur du workflow avec une limite de 5 injections par jour.

  • Pas besoin d'abonnement pour commencer
  • Import de skins custom compatibles
  • Organisation de la bibliothèque
  • Jusqu'à 5 injections par jour

Étape 4: importer vos skins custom League of Legends

Une fois l'application ouverte, ajoutez vos skins compatibles, organisez votre bibliothèque, sélectionnez votre champion, puis préparez votre workflow avant l'injection.

Bibliothèque claire

RIFTSKIN centralise vos skins par champion pour éviter un workflow dispersé.

Import puis lancement

Vous importez, vous préparez votre champion, vous injectez, puis vous lancez votre partie.

Quand passer au premium

Le premium devient utile si vous utilisez souvent RIFTSKIN et que la limite quotidienne gratuite vous ralentit.

  • Injections illimitées
  • Champions favoris
  • Skins favoris
  • Reconnaissance automatique du champion pendant le pick
