This guide explains how to install RIFTSKIN on macOS, how to start in free mode, how to import your custom skins, and when Premium becomes useful.

Step 1: download RIFTSKIN on Mac

Start by downloading the macOS installer from the official download page. This is the foundation of the workflow for using League of Legends custom skins on Mac with RIFTSKIN.

Download RIFTSKIN

Step 2: install the app

Open the installer, allow it in macOS if needed, then complete the installation. Once RIFTSKIN is inside Applications, you can launch it like any other Mac app.

  • Download the installer
  • Allow it in Privacy & Security if macOS blocks it
  • Finish the installation
  • Launch RIFTSKIN from Applications

Step 3: start in free mode

Free mode is enough to get started. It lets you import your custom skins, organize them, and use the core workflow with a limit of 5 injections per day.

  • No subscription needed to get started
  • Import compatible custom skins
  • Organize your library
  • Up to 5 injections per day

Step 4: import your League of Legends custom skins

Once the app is open, add your compatible skins, organize your library, select your champion, then prepare your workflow before injection.

Import, then launch
