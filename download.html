Download RiftSkin for macOS

Download the latest macOS build, install it, then choose free mode or activate premium directly inside the app.

Download and install on macOS

Use the macOS disk image for the cleanest install flow. Open the DMG, drag RiftSkin into Applications, then launch it from your Applications folder.

Download for macOS

Install flow

  • Download and open the RIFTSKIN-macos.dmg file
  • Drag RiftSkin.app into Applications
  • Launch RiftSkin from your Applications folder
  • On first launch, if macOS blocks the app, right click RiftSkin and choose Open
  • If needed, go to System Settings → Privacy & Security → Open Anyway
