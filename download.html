Guide d'installation visuel

Suivez l'installation sur macOS étape par étape.

Si macOS ralentit le premier lancement, ce guide visuel montre le parcours à suivre : télécharger le paquet, l'autoriser une fois, terminer l'installation, puis ouvrir RIFTSKIN depuis Applications.

01

Téléchargez l'installeur .pkg

02

Autorisez-le une fois dans Confidentialité et sécurité

03

Lancez l'installeur macOS

04

Ouvrez RIFTSKIN depuis Applications

Étape 01

Téléchargez le paquet d'installation

Commencez avec le bouton de téléchargement ci-dessus. Le paquet apparaît dans Téléchargements et vous pouvez l'ouvrir directement depuis ce dossier.

Étape 02

Autorisez le paquet dans les réglages de sécurité macOS

Si macOS bloque l'installeur, ouvrez Réglages système, allez dans Confidentialité et sécurité, puis autorisez RIFTSKIN avec Ouvrir quand même.

Étape 03

Confirmez l'autorisation de sécurité avec votre mot de passe administrateur

macOS peut vous demander votre mot de passe administrateur ou Touch ID juste après avoir cliqué sur Ouvrir quand même.

Étape 04

Relancez l'installeur

Une fois le paquet autorisé, rouvrez-le et poursuivez l'installation jusqu'à l'écran principal de l'installeur.

Étape 05

Confirmez l'autorisation finale de l'installeur

macOS peut demander une dernière confirmation par mot de passe ou Touch ID avant de copier RIFTSKIN dans Applications.

Étape 06

Terminez puis lancez RIFTSKIN depuis Applications

Quand l'écran de fin d'installation apparaît, RIFTSKIN est prêt dans Applications. Fermez l'installeur, lancez l'application, puis terminez la configuration dans le logiciel.
