Download and install on macOS

Use the macOS installer package for the cleanest setup. Open the package, complete the installer, then allow RiftSkin in macOS security settings if prompted.

Download for macOS

Install flow

  • Download and open RiftSkin-macos-installer.pkg
  • Follow the installer until RiftSkin is installed in Applications
  • Open Applications and launch RiftSkin once
  • If macOS blocks the app, close the alert and open System Settings > Privacy & Security
  • In the Security section, find the message saying RiftSkin was blocked and click Open Anyway
  • Confirm Open Anyway one more time if macOS asks again, then launch RiftSkin from Applications
