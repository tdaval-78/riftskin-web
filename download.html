Visual install guide

Follow the macOS install flow step by step.

If macOS slows down the first launch, this visual guide shows the clean path: download the package, allow it once, finish the installer, then open RiftSkin from Applications.

01

Download the .pkg installer

02

Approve it once in Privacy & Security

03

Run the macOS installer

04

Launch RiftSkin from Applications

Step 01

Download the installer package

Start from the download button above. The package appears in Downloads and you can open it directly from there.

Step 02

Approve the package in macOS security settings

If macOS blocks the installer, open System Settings, go to Privacy & Security, then allow RiftSkin with Open Anyway.

Step 03

Confirm the security approval with your administrator password

macOS can ask for your administrator password or Touch ID right after you click Open Anyway.

Step 04

Relaunch the installer

Once the package is allowed, reopen it and continue through the installer until you reach the installation screen.

Step 05

Confirm the final installer authorization

macOS can ask for one final password or Touch ID confirmation before it copies RiftSkin into Applications.

Step 06

Finish and launch RiftSkin from Applications

When the installation success screen appears, RiftSkin is ready in Applications. Close the installer, launch the app, then complete the in-app setup flow.

Premium activation guide

Enter your premium activation code inside the app.

There are two valid ways to activate Premium: during first-launch onboarding, or later from Settings in the app. In both cases, enter the same activation code and confirm premium access.

01

Use the onboarding access screen on first launch

02

Use Settings > Account later if you already continued in free mode

Option 01

Activate the code during onboarding

On the access setup screen, paste your premium code into the activation field, then click Activate key. You can continue directly with premium access from this first setup flow.

Option 02

Or activate it later from Settings

If you already continued in free mode, open Settings, switch to the Account tab, paste the same premium code, then click Activate key. This activates the same premium access afterward.
