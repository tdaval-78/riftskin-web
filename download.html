Guide d’installation macOS

Si macOS bloque l’installateur au premier lancement, suivez ce parcours visuel. L’installation reste simple : télécharger, autoriser, installer, puis ouvrir RIFTSKIN.

Étape 01

Téléchargez le fichier `.pkg`

Commencez par télécharger l’installateur depuis la page RIFTSKIN. Le fichier apparaît ensuite dans votre dossier Téléchargements.

Étape 02

Si macOS bloque le fichier, cliquez d’abord sur Terminé

Au premier essai, macOS peut afficher une alerte indiquant que le paquet n’a pas pu être vérifié. Fermez cette alerte pour pouvoir l’autoriser ensuite dans les réglages.

Étape 03

Autorisez l’ouverture dans Réglages système

Allez dans Réglages système > Confidentialité et sécurité, puis cliquez sur Ouvrir quand même pour autoriser le paquet.

Étape 04

Relancez l’installateur puis validez l’installation

Rouvrez le paquet, cliquez sur Install, puis utilisez Touch ID ou le mot de passe administrateur si macOS le demande.

Étape 05

Installation terminée

Une fois l’écran de confirmation affiché, l’application est installée. Vous pouvez fermer l’installateur puis lancer RIFTSKIN depuis Applications.
