Download for macOS FAQ

Download RiftSkin for macOS

Download the latest macOS build, install it, then choose free mode or activate premium directly inside the app.

Download and install on macOS

Use the macOS installer package for the cleanest setup. Open the package, allow it in macOS security settings if it is blocked, then complete the installer.

Download for macOS Open public releases
If macOS blocks the installer or the app on first open, that is expected for the current distribution flow. The security approval steps below are part of the normal setup.

Install flow

  • Download and open RiftSkin-macos-installer.pkg
  • If macOS blocks the installer, close the alert and open System Settings > Privacy & Security
  • In the Security section, find the message saying RiftSkin was blocked and click Open Anyway
  • Confirm Open Anyway one more time if macOS asks again, then reopen the installer package
  • Follow the installer until RiftSkin is installed in Applications
  • Open Applications, launch RiftSkin, and finish the in-app setup

Before you launch

RiftSkin is currently focused on macOS distribution. Once installed, you can stay in free mode or attach a premium code later from the desktop app.

Need help after install?

Use the support page if you hit a macOS security warning, import issue, activation problem, or launch regression after an update.
