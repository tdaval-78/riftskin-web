Frequently asked questions

Is the app usable for free?

Yes. Free mode keeps the core workflow available: import skins, browse the library, and run the game workflow.

What does premium unlock?

Premium unlocks favorite champions, favorite skins, and automatic champion recognition during pick phase.

Do I need to sign in inside the desktop app?

No. The desktop app works without a software login. Premium is activated with a code on the machine.

Do I still need the default skin in game?

Yes. For replacement to apply correctly, select the champion default/base skin in game.

How do I get premium?

Subscribe from the web account page. Your subscription stays monthly at 15 EUR, and you use the premium code in the app while the subscription is active.

Does RiftSkin support Windows?

Not in the current public distribution. The current supported workflow is focused on macOS.
