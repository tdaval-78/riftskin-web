Guide d'activation Premium

Renseignez votre code d'activation premium dans l'application.

Deux chemins fonctionnent pour activer Premium : pendant l'onboarding du premier lancement, ou plus tard depuis les paramètres de l'application. Dans les deux cas, entrez le même code d'activation puis validez l'accès premium.

01

Utilisez l'écran d'accès de l'onboarding au premier lancement

02

Ou utilisez Paramètres > Account plus tard si vous avez déjà continué en gratuit

Option 01

Activez le code pendant l'onboarding

Sur l'écran de choix d'accès, collez votre code premium dans le champ d'activation, puis cliquez sur Activate key. Vous pouvez continuer directement avec l'accès premium depuis ce premier setup.

Option 02

Ou activez-le plus tard depuis Paramètres

Si vous avez déjà continué en gratuit, ouvrez Settings, allez dans l'onglet Account, collez le même code premium, puis cliquez sur Activate key. Cela active le même accès premium ensuite.
